# Academic Portfolio - Dhiraj Patel

A professional academic portfolio website built with React, TypeScript, and Tailwind CSS. This site showcases research, publications, teaching experience, and professional achievements.

## Features

- **About**: Personal summary and core competencies.
- **Research**: Categorized publications and preprints with clear descriptions.
- **Conferences**: International conference presentations and workshop participations.
- **Teaching**: Detailed academic instruction experience.
- **Mentorship**: Records of student supervision at various academic levels.
- **Resume/CV**: A structured view of professional summary, education, and work experience.
- **Contact**: Integrated contact channels and location map.
- **Responsive Design**: Fully optimized for desktop and mobile devices.

## Tech Stack

- **React 18**: Frontend framework
- **TypeScript**: Static typing for robust code
- **Vite**: Modern build tool
- **Tailwind CSS**: Utility-first CSS for responsive styling
- **Framer Motion**: Fluid animations and micro-interactions
- **Lucide React**: Icon library
- **Google Maps Embed**: For office location

## Project Structure

- `src/components`: UI components organized by section (Home, Research, About, etc.)
- `src/data`: Centralized data configuration (Profile, Publications, Teaching, etc.)
- `src/lib`: Utility functions
- `public/`: Static assets including profile images

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
